# AI Spam Classifier - PRD

## 1. Goal
Build a model to classify messages as spam or not spam.

## 2. Users
- Email users
- Messaging platforms
- Businesses

## 3. Value
- Reduce spam messages
- Protect users from scams
- Save time

## 4. KPIs
- Accuracy ≥ 90%
- Precision (Spam) ≥ 95%

## 5. Scope
Included:
- Text classification (spam/ham)

Not included:
- Image spam detection
- Multilingual support (for now)

## 6. Risks
- False positives (blocking real messages)
- Data drift (new spam patterns)

## 7. Constraints
- Fast prediction (< 1 second)
- Low cost
