# Labeling Process

## Tool Used
Label Studio

## Task
Text classification (Spam vs Not Spam)

## Labels
- spam
- not spam

## Process
The dataset was uploaded to Label Studio.
Each message was annotated manually according to the defined schema.

## Guidelines
- Messages containing promotions, prizes, or offers → spam
- Normal communication messages → not spam

## Example
"Win a free iPhone" → spam  
"Meeting at 10am" → not spam
